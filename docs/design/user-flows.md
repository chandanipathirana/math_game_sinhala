# User Flows

## Child Flow

1. Open app
2. Tap `ආරම්භ කරන්න`
3. Select grade
4. Select topic
5. Complete five-question session
6. See immediate feedback after each answer
7. See summary
8. Replay or change topic

## Parent Flow

1. Open app
2. Tap `දෙමාපියන්`
3. Enter PIN
4. View progress summary
5. Review recent sessions and weak topics
6. Optionally reset progress
