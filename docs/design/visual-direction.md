# Visual Direction

## Direction

Bright, tactile, and warm. The interface should feel like a printed primary activity book turned into a touch-first app.

## Palette

- Warm cream background
- Coral primary action color
- Golden accent color
- Blue secondary chips
- Green success accents

## UI Rules

- Rounded cards
- Large touch targets
- No dense menus
- Minimal text per screen
- Motion limited to quick state transitions and progress changes
