# Progress Persistence

## Storage Keys

- `sinhala-math-progress`
- `sinhala-math-parent-pin`

## Session Entry Shape

- session id
- grade
- topic
- total questions
- correct answers
- stars
- completion timestamp

## MVP Choice

Local storage is sufficient for the first release because:

- it supports low-friction offline-friendly use
- it requires no backend setup
- it allows a parent dashboard immediately
