# math_game_sinhala

Sinhala primary math practice MVP built as a static web app.

The app now includes:

- student math practice flow
- parent progress dashboard
- admin question management interface

## Run

```bash
npm start
```

Open `http://localhost:4173`.

Admin access:

- Parent PIN: `1234`
- Admin PIN: `4321`

## Check content

```bash
npm run check
```
